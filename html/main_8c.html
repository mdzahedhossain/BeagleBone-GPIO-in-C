Detailed Description

Addressing multiple BeagleBone GPIO pins in C.

Self-contined program that will turn ON/OFF a specified group of of GPIO pins in the beaglebone board. This code was possible due to prior efforts by Christian Mitchell, Markus Klink and Matt Richardson. To compile and run this code, type something like "gcc main.c ; ./a.out ; rm -f a.out"
